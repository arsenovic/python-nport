.. _setup:

Installing nport
================

Requirements
------------

The :mod:`nport` package requires at least version 2.5 of Python.

As for Python packages, :mod:`nport` depends on numpy and scipy. :mod:`nport`
has been tested with numpy 1.4 and scipy 0.8, but older versions of these will 
likely work too.


Installation
------------

Installing :mod:`nport` can be done in the standard way::

    python setup.py install
